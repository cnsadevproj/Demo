# 네이버 블로그 글쓰기 스킬

충남삼성고등학교 생물 교사의 블로그 문체를 학습하여 네이버 C-RANK 및 DIA+ 알고리즘에 최적화된 교육 콘텐츠를 생성합니다.

## 핵심 기능

1. **4가지 글 유형 자동 감지**: 농생일지, 프로그램공유, 생물관찰, 교육도구
2. **해시태그 30개 자동 생성**: 마지막 4개는 #충남삼성고등학교,#충남삼성고,#삼성고,#CNSA로 고정
3. **이미지 배치 가이드**: `[이미지: 설명]` + 📷 캡션 형식
4. **SEO 최적화**: C-RANK(전문성), DIA+(경험 강조)

## 사용 방법

### 기본 사용
```
주제: [글 주제]
개요: [간단한 내용 설명]
```

### 상세 지정
```
주제: [주제]
유형: [농생일지/프로그램공유/생물관찰/교육도구]
개요: [내용]
특이사항: [강조할 점]
```

## 글 유형별 특징

### 1. 농생일지 (🌱🌻🥔)
- **문체**: "~했습니다", "~했어요" 혼용, 친근체
- **구조**: 시간순 전개, 활동별 소제목
- **특징**: 학생 활동 중심, 교육적 의미 강조
- **길이**: 1,800-2,200자

### 2. 프로그램공유 (🧪💻📊)
- **문체**: "~합니다", "~할 수 있습니다", 설명체
- **구조**: 개발배경 → 기능설명 → 사용법(단계별) → 링크
- **특징**: 기술적 설명 + 교육 목적
- **길이**: 2,000-3,000자

### 3. 생물관찰 (🦋🐛🕷️)
- **문체**: 회고적, 관찰 일기형
- **구조**: 만남 → 관찰 → 의미
- **특징**: 개인 경험 + 생물학적 설명
- **길이**: 1,200-1,500자

### 4. 교육도구 (📱🎓✏️)
- **문체**: 실용적, 안내형
- **구조**: 고민 → 발견 → 활용법 → 성과
- **특징**: 실제 사례 중심
- **길이**: 2,500-3,500자

## 문체 핵심 요소

### 톤앤매너
- 전문성과 친근함의 균형
- 학생 중심 시각 유지
- 과정 중심 기록(성공/실패 모두)

### 문장 구조
- 짧은 문단 (3-5줄)
- 명확한 소제목 구분
- 이모지 적절히 활용

### 필수 표현
- 학생 지칭: "아이들", "녀석들", "학생들"
- 식물 의인화: "친구들", "~해주었습니다"
- 교육적 가치: "~을/를 통해 배울 수 있었습니다"

## 해시태그 생성 규칙

### 구조 (총 30개)
- **주제 키워드** (15개): 글의 핵심 내용
- **교과 연계** (5개): #생물교육 #통합과학 #농업생명과학 등
- **계절/시기** (3개): #3월 #봄 #파종기 등
- **활동 유형** (3개): #수업활동 #실험실습 #관찰기록 등
- **고정 학교명** (4개): #충남삼성고등학교,#충남삼성고,#삼성고,#CNSA

### 형식
```
#키워드1,#키워드2,...,#충남삼성고등학교,#충남삼성고,#삼성고,#CNSA
```

## 이미지 처리

### 표기 방법
```markdown
[이미지: 옥수수 발아 모습]
📷 캡션: 포트에 심은 옥수수가 C자 훅을 만들며 발아하는 모습
```

### 배치 원칙
- 도입부 후 1장 (전체 모습)
- 각 활동/단계마다 1-2장
- 중요 설명 뒤 배치
- 캡션은 15자 내외로 간결하게

## SEO 최적화 전략

### C-RANK 최적화
- **Context**: 생물교육 키워드 집중
- **Content**: 전문 용어 + 쉬운 설명
- **Chain**: 질문 유도, 다음 활동 예고
- **Creator**: 교사 전문성, 학교명 일관성

### DIA+ 최적화
- **검색 의도**: "어떻게", "방법", "사용법" 키워드
- **경험성**: "직접 ~했습니다", "학생들이 ~했어요"
- **독창성**: 개인 경험 + 전문 지식, 실패 사례도 포함
- **체류시간**: 적절한 문단 길이, 소제목 구조, 이미지 배치

## 금지 사항

### 절대 금지
- ❌ 학생 이름 명시 (개인정보)
- ❌ 과도한 광고성 문구
- ❌ 정치적 발언
- ❌ 검증 안 된 과학 정보

### 주의 사항
- ⚠️ 학생 사진은 "[학생 활동 모습]"으로 표기
- ⚠️ 실험 결과 정확하게
- ⚠️ 출처 있는 정보 명시
- ⚠️ 안전 주의사항 필수
